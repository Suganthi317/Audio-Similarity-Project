# Audio Similarity & Frequency Comparison

This project compares two audio recordings and gives a similarity score.

## Features
- FFT frequency graph for both audios
- MFCC similarity
- Spectral centroid similarity
- RMS energy similarity
- Final similarity score (%)

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Put your audio files inside audio/ folder:
   audio1.mp3
   audio2.mp3

3. Run:
   python main.py
